# DemandAndSupply
# AnswerToRequest
The purpose of the application is to generate Answer after receiving the Request.

Request contains specific requirements for production: Scope, Time, Price, Priority.
Answer contains certain number of Proposals.
Proposals are generated from Producers, which meets the requirements of Request.


application will be developed in Stages.
Stage 1: Scope and Time requirements will be met.

Stage 2: Price and Priority requirements will be met.

Stage 3: ClientDatabase will be created, client will be added to it after receiving the Request.

Stage 4: Another module will be created to handle ProducersDatabase. After receiving the Producer data  it will be added to ProducersDatabase.
